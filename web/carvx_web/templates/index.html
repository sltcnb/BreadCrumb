What do these options do?
Offset / Length
Restrict the scan to a region — skip boot sectors or target a known partition.
Alignment
Only accept headers at N-byte boundaries. Higher = faster, fewer false positives, may miss embedded files.
Deep validate
Full-decodes each carve (JPEG/PNG/ZIP/SQLite/gzip…) and tags it verified or failed.
Dry run
Emits the manifest (offsets, sizes, hashes) without recovering files — quick assessment.
